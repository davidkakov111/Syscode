# SyscodeCli

To run this Angular application, you'll need the following:

1.  **Node.js and npm (or yarn):**
    * Angular, and the npm packages it relies on, require Node.js.
    * npm (Node Package Manager) is bundled with Node.js.
    * Download and install the latest LTS (Long Term Support) version of Node.js from [nodejs.org](https://nodejs.org/). This will also install npm.
    * Alternatively, you can use yarn, if you prefer. Yarn can be installed from [yarnpkg.com](https://yarnpkg.com/).

2.  **Angular CLI (Command Line Interface):**
    * The Angular CLI is essential for creating, developing, and managing Angular applications.
    * Install it globally using npm:
        ```bash
        npm install -g @angular/cli
        ```
    * Or using yarn:
        ```bash
        yarn global add @angular/cli
        ```

3.  **Install dependencies:**
    ```bash
    npm install
    ```

4.  **Run the development server:**
    ```bash
    ng serve
    ```

5.  **Navigate to `http://localhost:4200/`.**
